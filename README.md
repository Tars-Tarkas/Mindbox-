# Тестовое задание Frontend junior в Mindbox

### Для запуска проекта

#### Клонируем проект

```
https://github.com/Tars-Tarkas/Mindbox-.git
```

#### Устанавливаем зависимости

```
npm install
```

#### Запускаем проект

```
npm run start
```

#### Тестирование проекта

```
npm run test
```

В проекте использовалось:

- ReactJS
- Typescript
- Ant Design

приятного просмотра!
